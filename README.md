# Bandit-Blu2e
# Bandit Level0 → Level1

## 목표
- SSH 접속 연습
- 기본 명령어(ls, cat, cd, file,. du, find) 익히기
- 다음 레벨 비밀번호 찾기

## Level0
- SSH 접속 : 'bandit0@bandit.labs.overthewire.org -p 2220'
- 비밀번호 입력
- 명령어 : 'cat readme' → Level1 비밀번호 확인

## Level1
- SSH 접속 : 'bandit1@bandit.labs.overthewire.org -p 2220'
- Level0에서 찾은 비밀번호 입력
- 명령어 연습 :
  - 'ls', 'cat readme', 'pwd', 'cd folder', 'file somefile', 'du -h somefile', 'find . -name "filename"'
- (전공도 아니고 배운적도 없고 Over The Wire : Bandit 짱짱뉴비의 입문기, 매우 힘들었음 사실 레벨1 접속하고 막막했음, cmd 복사도 처음 알았음)
